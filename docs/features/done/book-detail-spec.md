# 书籍详情 (Book-Detail) Spec

## 要解决的问题
用户在书架或书城中点击一本书后，需要查看该书的详细信息（封面、书名、作者、评分、简介、目录），并决定是否加入书架、下载或开始阅读。

## 验收标准
- [ ] 进入页面时正确显示书籍信息：封面、书名、作者、评分、阅读人数
- [ ] 显示书籍简介，支持展开/收起（简介过长时）
- [ ] 显示目录列表（默认显示前2章），点击"查看全部"跳转目录页
- [ ] 点击章节可跳转到对应章节阅读（本期显示 Toast）
- [ ] 点击"加入书架"按钮，书籍添加到书架，按钮状态变为"已在书架"
- [ ] 点击"下载"按钮，显示下载进度（本期显示 Toast）
- [ ] 点击"立即阅读"按钮，跳转到阅读器页面并传入正确的 bookId
- [ ] 点击返回按钮，返回上一页
- [ ] 点击分享按钮，分享书籍（本期显示 Toast）

## 交互说明
- 从书架列表点击书籍 → 进入书籍详情页
- 从书城点击书籍 → 进入书籍详情页
- 点击"查看全部目录" → 跳转到目录页（Read-Contents）
- 点击"立即阅读" → 跳转到阅读器页面（Reader）

## 不在本期范围内
- 下载功能（仅显示 Toast 提示）
- 分享功能（仅显示 Toast 提示）
- 章节跳转阅读（仅显示 Toast 提示）
- 书籍评论/书评功能
- 相似书籍推荐

## 技术约束
- 数据来源：Room（本地书籍）+ Mock（书籍详情数据）
- 复用组件：无
