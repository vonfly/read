# 阅读器 (Reader) Spec

## 要解决的问题
用户需要阅读已添加到书架的书籍，支持翻页、调整阅读设置（字体、主题、亮度）、查看目录、保存阅读进度等核心阅读功能。

---

## 第一期：阅读器核心（MVP）

### 验收标准
- [ ] 从书籍详情页点击"立即阅读"跳转到阅读器页面
- [ ] 正确渲染书籍内容（段落文本）
- [ ] 左右滑动或点击屏幕左右区域可翻页
- [ ] 顶部显示章节标题，底部显示阅读进度（当前页/总页数）
- [ ] 点击屏幕中央显示/隐藏控制栏（TopBar + BottomBar）
- [ ] 阅读进度自动保存到本地，下次打开恢复到上次位置
- [ ] 返回按钮正常工作，返回书籍详情页

### 交互说明
- 点击屏幕左 1/3 区域：上一页
- 点击屏幕右 1/3 区域：下一页
- 点击屏幕中央 1/3 区域：显示/隐藏控制栏
- 左右滑动：翻页（与点击翻页二选一或共存）
- 返回按钮：返回书籍详情页

### 不在本期范围内
- 翻页动画（仿真翻页、滑动翻页等）
- 阅读设置（字体大小、主题切换）
- 目录功能
- 书签功能
- 网络书籍获取（仅 Mock 数据）

### 技术约束
- 数据来源：Mock（模拟书籍内容）
- 文本渲染：LazyColumn + 分页
- 翻页：HorizontalPager（Compose Foundation）
- 进度保存：Room

---

## 第二期：阅读设置

### 验收标准
- [ ] 底部控制栏显示"字体"、"亮度"按钮
- [ ] 点击"字体"弹出字体设置面板（字号、行距）
- [ ] 字号调整实时生效（范围 12sp - 32sp）
- [ ] 行距调整实时生效（范围 1.5 - 2.5）
- [ ] 点击"亮度"弹出主题设置面板（多种主题色）
- [ ] 支持 5 种主题：默认（米白）、夜间、护眼（米黄）、绿色、粉色
- [ ] 阅读设置自动保存，下次打开保持上次设置

### 不在本期范围内
- 自定义主题（用户自选颜色）
- 字体选择（仅使用系统默认字体）

### 技术约束
- 数据来源：DataStore（reader_settings）
- 主题管理：CompositionLocal

---

## 第三期：目录与书签

### 验收标准
- [ ] 底部控制栏显示"目录"按钮
- [ ] 点击"目录"弹出章节列表面板
- [ ] 点击章节跳转到对应位置
- [ ] 阅读时显示当前章节标题

### 不在本期范围内
- 书签功能（下一期）
- 章节缓存

### 技术约束
- 数据来源：Mock（模拟章节列表）

---

## 第四期：书签与笔记

### 验收标准
- [ ] 长按文本可选择文字
- [ ] 选择文字后可添加书签或划线
- [ ] 书签列表可在目录中查看
- [ ] 点击书签跳转到对应位置

### 不在本期范围内
- 笔记功能
- 云同步

---
